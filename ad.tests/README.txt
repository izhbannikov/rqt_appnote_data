This folder contains results on gene-level association study of Alzheimer's disease and 
consists of the following folders and files:

- 'rqt.analysis' - contains results of rqt gene level tests
- 'skat.analysis' - contains results of SKAT method
- 'skat.o.analysis' - contains results of SKAT-O method

File AD_genes_GWAS_catalog_NHGRI_June2016.xlsx contains SNPs and genes 
from NHGRI which are likely to be involved to Alzheimer's disease (AD).

Files genes.txt and genes.xlsx contain list of genes tested on associations to AD.
