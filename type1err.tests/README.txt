This folder contains results for type 1 error rates estimations of RQTests, QTests, 
SKAT and SKAT-O.

Folders:

- 'QQ_plots' - contains scripts and Q-Q plots.
- 'cont' - contains scripts and results for continuous outcome under null hypothesis.
- 'dich' - contains scripts and results for dichotomous outcome under null hypothesis.
Files:

- 'barplots_lasso.R', 'barplots_pca.R', 'barplots_pls.R', 'tables.R' - are scripts to 
make corresponding barplots.
