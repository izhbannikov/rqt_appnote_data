This repository contains supplementary data and scripts for the applications note 
"rqt: an R package for gene-level meta-analysis" and consists of the following folders:

1. 'ad.tests' - contains results on gene-level association study of Alzheimer's disease.
2. 'power.tests' - contains simulation results on power estimation.
3. 'type1err.tests' - contains simulation results on type 1 error rates.
